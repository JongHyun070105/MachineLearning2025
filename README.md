# 머신러닝 기초 실습 (2025)

이 저장소는 머신러닝 기초 수업의 실습 내용을 포함합니다. Python의 데이터 분석 라이브러리인 Numpy와 Pandas를 활용한 데이터 처리 및 분석 기법을 학습합니다.

## 주요 학습 내용

### 1. Numpy 기초
- NumPy 배열 생성 및 조작
- 배열 연산 및 수학적 연산
- 인덱싱 및 슬라이싱
- 통계 함수 활용

### 2. Pandas 기초
- Series와 DataFrame 다루기
- 데이터 인덱싱 및 선택
- 데이터 필터링 및 정렬
- 데이터 집계 및 그룹화
- 피벗 테이블 활용

### 3. 데이터 전처리
- 수치형 데이터 전처리
- 범주형 데이터 전처리
- 데이터 병합 및 연결
- 데이터 구조 확인 및 변환
